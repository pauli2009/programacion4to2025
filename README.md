# Programación Asociada a Python

Este repositorio contiene los materiales y ejemplos relacionados con la materia de **Programación Asociada a Python**.

## Contenidos

- Introducción a Python
- Variables y tipos de datos
- Control de flujo (condicionales y bucles)
- Funciones y módulos
- Estructuras de datos (listas, tuplas, diccionarios, conjuntos)
- Programación orientada a objetos
- Manejo de errores y excepciones
- Archivos y entrada/salida
- Uso de librerías externas (como NumPy, pandas, etc.)
- Proyectos y ejercicios prácticos

## Objetivos

El objetivo principal de este curso es aprender a programar en Python, comprender su sintaxis y paradigmas, y aplicarlos en proyectos reales y ejercicios prácticos.

## Cómo usar este repositorio

- Revisa los archivos y ejemplos en cada sección.
- Participa en los ejercicios y proyectos.
- Contribuye con mejoras o nuevas ideas.
